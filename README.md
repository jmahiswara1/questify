# Questify

Questify is a modern, interactive web-based trivia application that challenges users with a variety of questions from the Open Trivia Database (OpenTDB).

## Tech Stack
- React
- Vite
- Vanilla CSS
- OpenTDB API

## Features
- Customizable Quizzes (Select category, difficulty, number of questions)
- Interactive Gameplay
- Detailed Results and Review
- Modern UI (Glassmorphism, Dark/Light Mode Theme Toggle)
